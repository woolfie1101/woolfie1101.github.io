포트폴리오 `resume.json`의 경력 프로젝트 데이터로 정리한 실제 작업 기록입니다.

## 프로젝트 개요

- 소속: 세일링스톤
- 역할: 풀스택 개발자
- 기간: 2023.11 - 2024.08
- 기간 시작일(커밋 기준): 2023-11-01

## 세부 수행 내용

- SKON 대시보드 웹 애플리케이션 유지보수 및 기능 개선
- 화면 단위 API 연동 구조를 정리해 오류 추적 포인트를 단순화
- 지표 조회 로직을 조정해 사용자 조작 시 지연 구간을 감소
- 운영자 입장에서 자주 쓰는 필터/정렬 조합의 기본값을 정리해 조회 효율 개선

## 사용 기술

Spring, JavaScript, HTML, jQuery
