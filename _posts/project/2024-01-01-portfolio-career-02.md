포트폴리오 `resume.json`의 경력 프로젝트 데이터로 정리한 실제 작업 기록입니다.

## 프로젝트 개요

- 소속: 세일링스톤
- 역할: 풀스택 개발자
- 기간: 2024.01 - 2024.12
- 기간 시작일(커밋 기준): 2024-01-01

## 세부 수행 내용

- 사내 플랫폼을 **Spring Boot + React/Next.js 13**로 처음부터 설계·개발
  - 마케팅 데이터 조회, 정합성 확인, 운영자 액션 기준 화면까지 전 구간을 풀스택 관점에서 설계
- 대규모 데이터 처리 개선 실험
  - Campaign/Ad Group/Keyword 단위 처리 시 OOM 위험 구간 점검
  - 브라우저 중심 처리에서 서버/배치 중심 처리로 분리해 안정성 확보 방향을 반영
- Google Ads 데이터 운영을 파일다운로드 중심에서 웹 기반 조회·분석 흐름으로 전환
- 내부/외부망 배포 경로를 정비하면서 배포 자동화와 이력 관리 포인트를 정리

## 사용 기술

Spring Boot, React, Next.js 13, Tailwind CSS
