포트폴리오 `resume.json`의 경력 프로젝트 데이터로 정리한 실제 작업 기록입니다.

## 프로젝트 개요

- 소속: 세일링스톤
- 역할: 풀스택 개발자
- 기간: 2025.01 - 현재
- 기간 시작일(커밋 기준): 2025-01-01

## 세부 수행 내용

- LG 로고 HS 마케팅 데이터 관리 웹 애플리케이션의 운영 안정성 개선
  - 캠페인·광고그룹·키워드 단위 대시보드 조회 성능 튜닝
  - 기존 엑셀 다운로드 중심 작업을 웹 기반 검증/조회 중심으로 전환하는 방향 정리
- PostgreSQL/SQLite 혼합 환경에서 Airflow 배치 워크플로우를 운영해 정기 적재와 리포트 생성 자동화
- 배치 실패 처리와 예외 케이스를 구분해 알림/재처리 루프를 정비
- 내부 승인 프로세스에 맞춘 운영 문서화와 간단한 모니터링 규칙 정리

## 사용 기술

Django, Python, JavaScript, jQuery, PostgreSQL, SQLite, Airflow
