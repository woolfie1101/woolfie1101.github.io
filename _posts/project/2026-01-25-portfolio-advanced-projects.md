## AI 제품 이미지 생성기 Pro

이커머스/홈쇼핑 업계를 대상으로 제품 제작 비용과 제작 시간을 크게 줄이는 방향으로 고도화 중인 AI 이미지 생성기입니다.
- 링크: [https://ai-clothing-image.vercel.app/](https://ai-clothing-image.vercel.app/)

### 개선 방향

- 사용자의 입력값을 기반으로 템플릿·스타일·사양 조합을 고도화해 반복 촬영/디자인 의존도를 낮춤
- 랜딩 페이지 UX 및 생성 옵션을 정비해 제작 플로우 진입장벽을 낮춤
- 생성 결과물의 품질 편차를 낮추기 위해 프롬프트/후처리 기준을 정량적으로 관리

## AI 코딩 테스트 마스터

Anki 망각 곡선 기반으로 복습 타이밍을 설계한 코딩 테스트 학습 서비스입니다.
- 링크: [https://coding-test-two-ecru.vercel.app/](https://coding-test-two-ecru.vercel.app/)

### 핵심 기능

- Python 핵심 개념 요약 자동 생성
- 한글/영문 해설 입력 시 LLM 피드백으로 정합성 검증
- 실전 학습 흐름에 맞춘 단계별 난이도 조절

## 스마트 쉐도잉 영어 학습 앱

실제 사용자의 학습 시나리오를 기준으로 설계한 영어 쉐도잉 앱입니다.
- 링크: [https://english-study-app-889563220811.asia-northeast3.run.app](https://english-study-app-889563220811.asia-northeast3.run.app)

### 구현 포인트

- 자주 쓰는 회화 문장 학습용 MP3 기반 반복 듣기/녹음/피드백 흐름
- Next.js 기반 클라이언트와 Flutter WebView 조합으로 모바일 환경에 맞춘 반응형 흐름
- 개인 학습 데이터 기반으로 학습 반복 구조를 강화
